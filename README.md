# How to use  
## **Drag & Drop folder**

Filter all files include 'Hello' or 'world'
> Hello|world  

Filter all files include 'Hello' and 'world'
> Hello&world  

Open file
> Mouse Right Click

Copy a selected filename
> Ctrl + C

Delete a selected filename
> Ctrl + D

Toggle show/hide file path
> Ctrl + H

Open Notepad
> Ctrl + M

Open folder of selected file
> Ctrl + O

Open MsPaint
> Ctrl + P

Quit
> Ctrl + Q
