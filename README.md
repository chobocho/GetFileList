# GetFileList
Get file list

![screen shot](https://github.com/chobocho/GetFileList/blob/master/ux/screenshot.png)

# How to use  
Filter all files include 'Hello' or 'world'
> Hello|world  

Filter all files include 'Hello' and 'world'
> &Hello&world  
* And must start with '&'
